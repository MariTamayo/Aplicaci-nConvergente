Backend API REST con FastAPI

API REST desarrollada con FastAPI que proporciona servicios backend. Incluye:

Autenticación y Registro de Usuarios: Sistema de registro con validación de contraseñas, verificación de emails duplicados y encriptación con bcrypt.
Gestión de Base de Datos: Integración con PostgreSQL usando SQLAlchemy ORM para modelar y persistir datos de usuarios.
CORS Habilitado: Configuración para permitir conexiones desde aplicaciones cliente (React Native, etc.).
Rutas de API Modularizadas: Endpoints organizados en módulos para operaciones relacionadas a usuarios.
Tecnologías: FastAPI, SQLAlchemy, PostgreSQL, Uvicorn, Passlib + Bcrypt
